Modeling and Control of a Quadruped Robot

This was the final year project for my B.Tech degree. The project had three main goals:

  • Develop a kinematic model of the quadruped robot.
  • Develop gait functions for the robot for stable walking
  • Develop a method to estimate the state of the robot using a minimal amount of sensors.

These three goals were successfully completed.

  • Guided by: Dr. E.S. Gopi, NIT Trichy
  • Time: Jan - April 2016
  • Area: Robotics

Design and Testing of a Bio-Inspired Quadruped

A project on designing and building a bio-inspired quadruped (four legged) robot with compliant limbs. This was carried out at NUS in two months as part of my summer internship. The goal of the project was to develop a quadruped robot with limbs that are compliant - like in real animals - which would allow the robot to be more robust and withstand shocks.

  • Guided by: Dr. Yu Haoyong, NUS
  • Time: May-July 2015
  • Area: Robotics

Solar Powered Electric Wheelchair with Robotic Arms

This project was carried out as part of a project competition held by Texas Instruments India (The Texas Instruments India Innovation Challenge). The goal was to build an electric wheelchair that could use solar energy to recharge itself. The wheelchair also had robotic arms that could help the user of the wheelchair pick up and move objects.

  • Guided by: Prof. Dr. Arul Daniel, NIT Trichy
  • Time: Dec 2014 - April 2015
  • Area: Robotics

Modeling and Control of a 3 Link Robotic Arm

The goal of this project was to control the position of the end effector of a 3 link robotic arm using image processing. The arm was modeled using the manipulator equations to calculate and compensate for the effects of gravity on the arm. A PID controller was used to control the gravity compensated joints' positions.

  • Guided by: Dr. G. Saravana Ilango, NIT Trichy
  • Time: May 2014
  • Area: Robotics

Planar 2-Link Robotic Arm that can Draw Shapes

A simple 2 link robotic arm I made that can draw shapes on paper. The motors used were AX-12A dynamixels. They were controlled using a python script with the pypot library.

Robust Line Following Using IR LEDs

A robust line following robot. The sensor was an array of four IR LED - Photodiode pairs that detected a black line. But getting this working was only half the battle. The code required hours of tweaking before it followed the line without stopping unexpectedly or loosing track of the line.

Laptop Music Control Using Telgram Bot

The Telegram messenger has a pretty awesome API to create bots. I used it to create a bot that runs on my laptop and can control the music on it. I can send the bot messages to play, pause and control the volume of the music. It can even do basic searches so that commands like "Play Adele" "Play Viva La Vida" work.
